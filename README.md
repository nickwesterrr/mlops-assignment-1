# ML Programming and MLOps Assignment 1

This repository contains my solutions for Assignment 1 of the course "ML Programming and MLOps".

## Installation

This project runs on the Snellius supercomputer.
Software is managed using environment modules and a Python virtual environment.

```bash
# Load required modules
source source /home/"user"/modules.sh

# Create and activate a virtual environment
uv venv venv
source ./venv/bin/activate

# Install required packages
uv pip install torch
```
